Heading and Text Test

Headings

  • Headings create the document outline
  • Headings levels should not ever be chosen for how they look
  • Headings always go in order when decending. For example, H4 should never follow an H2. H4 comes after H3 or other H4 elements.
  • Only use one H1 per page even though HTML 5 spec says you can use more.
  • Headings should rarely come one after the other with no other content between them

Examples of Proper Heading Order

  • H1
    • H2
    • H2
      • H3
        • H4
        • H4
    • H2
