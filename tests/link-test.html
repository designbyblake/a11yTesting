Default Links

Good and bad text examples

Links should have descriptive text and avoid repetition of the same text over and over again. Phrases to avoid are click here, here, more, read more, learn more, etc.

Title Attribute

It is best practice to not put relevant information only in a title attribute. Some assitive technologies will read titles some do not. Avoid repetion in the title attribute and link text as both may be read.
